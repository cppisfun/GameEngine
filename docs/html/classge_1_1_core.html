Ausführliche Beschreibung

Hauptkomponente von GameEngine, verwaltet alle Kernobjekte.

Wurde nach dem Singleton-Muster entworfen und steht somit immer durch Aufrufen von Core::Instance() zur Verfügung.

Die Klasse instantiiert Kernobjekte entweder durch Aufrufen der Methode Reset() oder Verwenden eines entsprechenden Getters. Sollte die Verwendung einer Kernkomponente vom Bestehen einer anderen abhängen, wird dies intern entsprechend berücksichtigt.
