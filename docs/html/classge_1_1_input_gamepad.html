bool ge::InputGamepad::ButtonPressed ( int  btn = ANY)

Ermittelt, ob gerade ein Button gedrückt wurde.

Im Gegensatz zu Button() ist dieser Status nur im Moment des Herunterdrückens true, nach dem nächsten Aufruf von Update() allerdings wieder false.
