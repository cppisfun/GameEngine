Im Gegensatz zu ButtonPressed() ist dieser Status immer true, solange die Taste gehalten wird.

Parameter
btnButton_None = keine, Button_Any = beliebig; Nummer 0 - 31 (Standard: Button_Any)
bool ge::InputGamepad::ButtonPressed ( int  btn = Button_Any)

Ermittelt, ob gerade ein Button gedrückt wurde.

Im Gegensatz zu Key() ist dieser Status nur im Moment des Herunterdrückens true, nach dem nächsten Aufruf von Update() allerdings wieder false.
