Ermittelt, ob eine Taste gehalten wird.

Im Gegensatz zu KeyPressed() ist dieser Status immer true, solange die Taste gehalten wird.

Parameter
keyKey_None = keine, Key_Any = beliebig; Nummer 0 - 255 (Standard: Key_Any)
bool ge::InputKeyboard::KeyPressed ( int  key = Key_Any)

Ermittelt, ob gerade eine Taste gedrückt wurde.

Im Gegensatz zu Key() ist dieser Status nur im Moment des Herunterdrückens true, nach dem nächsten Aufruf von Update() allerdings wieder false.
